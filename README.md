Este projeto foi desenvolvido em conjunto com Daniel com o objetivo de aprofundar nossos conhecimentos sobre os componentes do Angular.

Implementamos um campo de pesquisa (Search) e um campo de seleção (Dropdown), utilizando o arquivo db.json para simular um banco de dados onde seriam enviadas as informações dos produtos. Além disso, criamos a funcionalidade de edição de produtos e documentamos todo o processo, incluindo os códigos utilizados.

O projeto foi estruturado para simular o uso do Angular Material, proporcionando uma referência prática que podemos consultar e reutilizar os códigos posteriormente.
